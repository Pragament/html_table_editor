# 📊 GridQuest: Kids Spreadsheet Lab

Welcome to **GridQuest**, a premium, interactive, and gamified spreadsheet application designed specifically for **Class 5 and 6 computer science students**. 

GridQuest transforms traditional, dry spreadsheet tables into an engaging learning sandbox. It aligns directly with the digital literacy syllabus, teaching students key concepts through guided missions, real-time feedback, and visually rewarding achievements.

---

## 🚀 Features

### 1. 🎓 Interactive Learning Academy (Sidebar)
*   **6 Guided Missions**: Walks students through cells, styling, arithmetic calculations, Range formulas (`=SUM`, `=AVERAGE`), data sorting, and building a student report card.
*   **XP & Leveling System**: Students gain XP upon completing mission conditions, climbing levels from "Robo-Cadet" to "Spreadsheet Master."
*   **Confetti & Badges**: Triggers confetti and unlocks visual badges on their academy shelf for achieving goals.

### 2. 🎨 Kid-Friendly Premium Aesthetics
*   **4 Theme Settings**:
    *   `Classic Emerald`: Familiar green office theme.
    *   `Cosmic Space`: Indigo and violet dark theme.
    *   `Bubblegum Candy`: Sweet pink and cyan light theme.
    *   `Cyberpunk`: Cybernetic dark theme with bright neon details.
*   **Avatar Selection**: Cycle through fun avatars: 🤖 Robo, 🦕 Dino, 🧙 Wizard, and 🧑‍🚀 Astro.
*   **Outfit Typography**: Premium, rounded modern layout.

### 3. 📈 Interactive SVG Charting Engine
*   Create dynamic **Bar**, **Line**, or **Pie** charts.
*   Fully draggable and resizable SVG graph windows that float on the grid workspace.
*   **Live Updates**: Graphs monitor cell data ranges and redraw in real-time as values are edited!

### 4. 📶 High-Performance Sorting Engine
*   Allows sorting selected columns or ranges ascending (A-Z / Min-Max) or descending (Z-A / Max-Min).
*   Correctly shifts corresponding row indices to keep dataset rows together.

### 5. ⚙️ Upgraded Spreadsheet Parser & Bug Fixes
*   **Full Formulas**: Evaluates standard arithmetic expressions with cell coordinates, e.g. `=A2+B2`, `=(A2-B2)/3`.
*   **Range Functions**: Supports recursive `=SUM(range)`, `=AVERAGE(range)`, `=MIN(range)`, `=MAX(range)`, and `=COUNT(range)`.
*   **Absolute References**: Supports cell anchors ($A$1) and toggle states (F4).
*   **Bulletproof Undo/Redo**: Implements deep snapshot states for all actions (edits, sorting, structural column/row resizing, magic fill, autofill).
*   **Selection Anchor**: Shift-click selection expansion and contraction works smoothly.

---

## 🛠️ How to Play / Run

GridQuest has **zero dependencies** and operates completely in the browser offline!
1. Double-click or open **`index.html`** in any modern web browser.
2. Open the **Academy Panel** on the right side.
3. Click **"Load Template"** on Mission 1 to begin your spreadsheet journey!
